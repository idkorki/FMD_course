# FMD_course

Курсовая работа по дисциплине «Формализованные модели данных».

Проект представляет собой учебную систему управления данными, реализующую:
- реляционную базу данных (PostgreSQL),
- работу с JSON,
- аналитические запросы к базе данных,
- консольное приложение для взаимодействия с БД.

---

## Функциональность системы

- создание и инициализация базы данных;
- заполнение таблиц тестовыми данными;
- выполнение аналитических запросов;
- экспорт данных из базы в JSON;
- импорт данных из JSON в базу данных;
- консольный интерфейс управления.

---

## Используемые технологии

- Python 3.x
- PostgreSQL
- psycopg2
- JSON
- SQL
